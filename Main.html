Ambas contienen los siguientes atributos globales:

title: título del elemento (manten el cursor en el texto).

style: Permite especificar información de estilo

draggable: Especifica si el elemento se puede arrastrar o no

Etiquetas para incrustar contenido

img:

etiqueta "height" y "width": Un número de píxeles que indica el tamano que la imagen tomará cuando sea incrustada en el documento.

etiqueta "source": la ubicacion del archivo

video:

etiqueta "height" y "width": Un número de píxeles que indica el tamano que el video tomará cuando sea incrustada en el documento.

etiqueta "source": la ubicacion del archivo

etiqueta "controls": indica al navegador una interfaz para controlar el video

Etiquetas para tablas

th y td:

atributo "colspan: El número de celdas hacia la derecha que esta celda debería abarcar

Encabezado 1
Celda 1 Celda 2

etiqueta "rowspan": El número de celdas hacia abajo que esta celda debería abarcar.

Celda 3

Eiquetas para formularios

label e input:

atributo "for": Un identificador que coincide con el valor del atributo id del control con el que esta etiqueta está asociada.
